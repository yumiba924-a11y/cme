▶ ⑥ 税効果会計 知識中心 ~14問
なぜ存在する？
会計上の利益 ≠ 税務上の課税所得（ズレが発生） → このズレを調整してPLで税引前利益と税金費用を対応させる仕組み
ズレの分類
┌─────────────────────────────────────────────┐ │ 「将来、税務上も認められる日が来る？」 │ │ │ │ YES → 一時差異（税効果の対象） │ │ NO → 永久差異（対象外、スルー） │ └─────────────────────────────────────────────┘
一時差異の2タイプ
将来減算一時差異（今多く払ってる → 将来減る） → 繰延税金「資産」 → 例: 貸倒引当金超過、賞与引当金、事業税、 退職給付引当金、減価償却超過額 将来加算一時差異（今少なく払ってる → 将来増える） → 繰延税金「負債」 → 例: 特別償却、その他有価証券評価差額金（評価益）
永久差異（対象外）— この3つだけ覚える
・交際費の損金不算入 ・受取配当金の益金不算入 ・寄付金の損金不算入限度超過額
繰延税金資産は毎期末に回収可能性を判断する
将来減算一時差異 → 繰延税金「資産」（資と減の組合せ）
将来加算一時差異 → 繰延税金「負債」（負と加の組合せ）
賞与引当金 → 一時差異（永久差異ではない！税務上は支払時に損金）
その他有価証券評価差額金 → 純資産直入でも一時差異（永久差異ではない）

▶ ⑧ EPS / PER 計算中心 ~14問
EPS（1株当たり当期純利益, Earnings Per Share） ＝ 親会社株主に帰属する当期純利益 ÷ 期中平均発行済株式数 PER（株価収益率, Price Earnings Ratio） ＝ 株価 ÷ EPS BPS（1株当たり純資産, Book value Per Share） ＝ 純資産 ÷ 期末発行済株式数 PBR（株価純資産倍率, Price Book-value Ratio） ＝ 株価 ÷ BPS

▶ ㉑ 企業価値評価 DCF 計算多め ~61問
主要な評価手法
【DCF法】6ステップ ① FCF算出（営業CF − 設備投資）← 営業利益じゃない！ ② 割引率 = WACC ③ 企業価値(EV) = FCFの現在価値合計 + ターミナルバリュー ④ 株主価値 = EV − 有利子負債 ⑤ 1株当たり価値 = 株主価値 ÷ 発行済株式数 【DDM（配当割引モデル）】 株式価値 = D₁ ÷ (r − g) D₁=来期配当, r=要求利回り, g=成長率 【RIM（残余利益モデル）】 株式価値 = BPS + Σ(当期残余利益の現在価値) 残余利益 = 当期純利益 − 株主資本コスト × 期首自己資本
DCFの③→④：企業価値から負債を引くのが株主価値（順番注意）
その他の頻出フレームワーク
NPV（正味現在価値）＝ CF現在価値合計 − 初期投資 → 0以上なら投資OK IRR（内部収益率）＝ NPV=0となる割引率 → WACCより高ければ投資OK PPM（BCGマトリクス）＝ 市場成長率 × 相対シェアで4象限 5フォース、VRIO、コーポレートガバナンス

生産者理論
費用の種類： FC（Fixed Cost / 固定費用） VC（Variable Cost / 変動費用） TC（Total Cost / 総費用）= FC + VC AC（Average Cost / 平均費用）= TC ÷ Q AVC（Average Variable Cost / 平均変動費用）= VC ÷ Q MC（Marginal Cost / 限界費用）= ΔTC ÷ ΔQ 利潤最大化条件：MR = MC （MRを最大化するのではない！）
市場構造
完全競争：Price Taker（価格への影響力なし） 長期均衡 → 超過利潤ゼロ 独占：MR = MC で生産量決定、P > MC 寡占：クールノー競争（数量）/ ベルトラン競争（価格）
